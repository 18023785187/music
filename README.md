# 说明
    该项目仅供学习使用，不作任何商业用途

# 启动本地服务

    $ git clone https://github.com/18023785187/music.git
    $ yarn install
    $ yarn start

    注意：
        默认启动的是https服务，如果需要使用http服务那么需要把 yarn start 改为 yarn start-http
        接口地址在 src/network/constant.ts -> BASE_URL

    在控制台调用window.testPerformance方法可以打印当前性能状况(效果如下)
